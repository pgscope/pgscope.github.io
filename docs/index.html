FAQs

What is PgScope?

PgScope is a tool to make it it nicer to use postgres databases.

How do I get started?

Simply create an account and start using PgScope. It's that easy!

How much does it cost?

It's free for personal cloud use and $2 per user for team cloud use. On-premise enterprise use is $6k per year regardless of the number of users.
